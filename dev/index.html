This is a Julia package for modeling and solving Newsvendor Problems.

Usage

The package is not yet listed and must be retrieved from the Github repo.

julia> import Pkg
julia> Pkg.add(url="https://github.com/frankhuettner/NewsvendorModel.jl")

After installation, it can be loaded with the usual command.

julia> using NewsvendorModel

Moreover, you need to load the distributions package.

julia> using Distributions

You can now define a model with the command NVModel(...) using the following required arguements (in that order):

  • unit production cost
  • unit selling price
  • demand distribution

Solve for optimal quanitity and obtain key metrics with the optimize() function.

Example

Consider a scenario with unit cost = 1, unit price = 5, and a demand function that has normal distribution with mean = 90 and standard deviation = 30. The model is now defined and stored in the variable nvm as follows:

julia> nvm = NVModel(1, 5, Normal(90, 30))

Julia shows the model data:

Data of the Newsvendor Model
 * Unit cost: 1.000000
 * Unit selling price: 5.000000
 * Demand distribution: Normal{Float64}(μ=90.0, σ=30.0)
 * Unit salvage value: 0.000000
 * Unit lost goodwill: 0.000000
 * Fixed cost: 0.000000

Next, you can solve the model like so:

julia> res = optimize(nvm)

This gives the following output:

=====================================
Results of maximizing expected profit
 * Optimal quantity: 116
 * Expected profit: 317.99
=====================================
This is a consequence of:
 * Cost of underage: 4.00
 * Cost of ovderage: 1.00
 * The critical fractile: 0.80
-------------------------------------
Ordering q = 116 further yields
 * Expected sales: 86.80
 * Expected lost sales: 3.20
 * Expected left over: 29.20
 * Expected lost goodwill: 0.00
-------------------------------------

Moreover, you have stored the result in the varial res. Reading the data from the stored result is straight-forward:

julia> q_opt(res)
116
julia> profit(res)
317.9926337188953

And analogously, underagecost(res), overagecost(res), criticalfractile(res), sales(res), lostsales(res), leftover(res), lostgoodwill(res).

Salvage value, lost goodwill, and fixed cost

Let us assume that an unsold unit can be salvaged for 0.3. Further suppose that an unserved customer might not return and we also miss future profits of 2. Finally, suppose that the operations require a fixed cost of 100. The model is now specified as follows:

julia> nvm2 = NVModel(1, 5, Normal(90, 30), salvage = 0.3, badwill = 2, fixcost = 100)
julia> optimize(nvm2)
=====================================
Results of Maximizing Expected Profit
 * Optimal quantity: 128
 * Expected profit: 223.58
=====================================
This is a consequence of:
 * Cost of underage: 6.00
 * Cost of ovderage: 0.70
 * The critical fractile: 0.90
-------------------------------------
Ordering q = 128 further yields
 * Expected sales: 88.53
 * Expected lost sales: 1.47
 * Expected left over: 39.47
 * Expected lost goodwill: 2.93
-------------------------------------
